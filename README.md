# YellowLights.ai - Portfolio Site

Decision science portfolio and mini-application platform for Trace Hancock.

## Tech Stack
- React 19 + TypeScript
- Vite (build tool)
- Tailwind CSS (via CDN)
- Lucide React (icons)

## Local Development
```bash
npm install
npm run dev
```

## Deployment
Deployed on Vercel at yellowlights.ai

## Features
- **Laboratory**: Interactive DecisionLab diagnostic tool (launching Q2 2025)
- **Archive**: Project showcase grid
- **Journal**: Blog/writing samples
- **Inquiry**: Contact and engagement information
